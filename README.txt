Bidding War Wheel - Run Instructions

1. Extract the ZIP into a folder.
2. Open terminal in the project folder.
3. Run:
   npm install
   npm run dev
4. Open the local URL printed by Vite (usually http://localhost:5173) on your laptop browser.

Notes:
- This project is desktop/laptop optimized (non-responsive).
- To edit labels or weights, open src/components/Wheel.tsx and modify SEGMENTS and WEIGHTS arrays.
- Center text is intentionally left blank as a placeholder.
